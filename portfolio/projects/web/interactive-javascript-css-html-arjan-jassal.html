Overview

The Social Media Invasion is an interactive infographic designed to illustrate the growth of Facebook and Twitter from 2009 to 2014.

What I Did

Interactive Infographic, JavaScript, CSS3

The Idea

The idea behind The Social Media Invasion was to take data that would otherwise be boring, and make it engaging and fun to interact with.

The Code

var allCountry = []; // store all instances of Country()

function Country(name, fb, tw, pos) {
	this.name = name; // use as id of div
	this.fb = fb; // facebook size [2009, 2014]
	this.tw = tw; // twitter size [2009, 2014]
	this.pos = pos; // x,y position
}

I used a function constructor to create objects to represent each country. Each object has properties with unique values that represent sizes and positions for each div that will be created. 

allCountry[0] = new Country( "usa",    [85, 178], [12, 34], [270, 194] );
allCountry[1] = new Country( "uk",     [20, 36],  [1, 15],  [625, 143] );
allCountry[2] = new Country( "canada", [12, 19],  [1, 5],   [249, 140] );

These are some instances of Country() being stored in an array for later use. The value of the first property is used as the id for the div that will be generated later. The second and third values are arrays where the first number determines the height and width of the div for 2009, the second number is the height and width of the div for 2014. The first array is for Facebook and the second one is for Twitter. The last property stores the x and y values of the div that will be applied when the divs are generated. 

Country.prototype.createDiv = function() {
  var div = document.createElement('div');
  var container = document.getElementById('circle-container');
  
  div.id = this.name; // set id
  div.className = 'circle'; // set class

  div.style.left = this.pos[0]; // x position
  div.style.top = this.pos[1]; // y position

  container.appendChild(div); // insert into container div
}

This is the method that creates all of the divs and gives them id's, a class, and positions based on properties in the Country prototype. 


var countryDivs = [];
var circles = document.getElementById('circle-container').childNodes;

for (var i = 0; i < circles.length; i++) {
	countryDivs.push('#' + circles[i].id);
}

The purpose of this code is to store all of the id's of each div into an array of it's own. That way the array can be targeted easily with for the click events about to take place. 

fb09.onclick = function() {
  for (i = 0, countries = countryDivs.length; i < countries; i++) {
    $(countryDivs[i]).velocity({
      width: allCountry[i].fb[0], // size based on value of object property fb
      height: allCountry[i].fb[0],
      translateX: '-50%',
      translateY: '-50%',
      backgroundColor: '#3b5998',
    }, {mobileHA: true, duration: 450, easing: 'spring'});
    document.getElementById(allCountry[i].name).setAttribute('title', allCountry[i].fb[0] + 'M');
  }
};

This loop cycles through countryDivs[] and uses Velocity.js to animate the width and height based on the values of the specified property of each object in allCountry[]. I'm also assigning each div a title attribute and using CSS to style the attribute so that when you hover over a circle it displays a number.

The Process

I originally created this project primarily using CSS3 and jQuery's addClass() and removeClass() methods. Although this worked, it was an absurd amount of work to just create even one country. There would be 4 different CSS rules for each country and then 4 different addClass() and removeClass() instances in the script. This was ridiculously repetitive and inefficient, but at the time I didn't know any better.

After expanding my JavaScript knowledge I thought that a great way to exercise this knowledge would be to re-code this project. I decided it would be suitable to store all the values I needed in objects and then use for loops to iterate through each object that I have stored in an array. I also wanted to use Velocity.js to animate things because it uses JavaScript to animate, which is faster than CSS and jQuery.
